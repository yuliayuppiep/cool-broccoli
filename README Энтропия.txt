README Энтропия
Энтропия является мерой неопределенности или случайности в информационной системе. В коде рассчитывается энтропия исходной строки на основе вероятности появления отдельных символов в ней.

Пример 1: Строка с одинаковыми символами
let input = 'aaaaa';
В этом случае каждый символ в строке одинаков, что приведет к низкой энтропии.

Пример 2: Строка с разными символами и преобладающим символом
let input = 'abcdabcd';
Здесь строки содержит разные символы, но символ 'a' повторяется чаще других, что уменьшает энтропию.

Пример 3: Строка со случайными символами
let input = 'qwertyuiopasdfghjklzxcvbnm9';
В данном примере строки содержит случайные символы без повторений, что увеличивает энтропию.

Код для вычисления энтропии может быть полезен в криптографии, компьютерной безопасности, анализе данных и других областях. Он помогает определить уровень случайности или неопределенности в строке данных. Например, вычисление энтропии может помочь в оценке качества случайной генерации ключей шифрования, определении комплексности паролей или анализе структуры текстовых данных. Высокая энтропия указывает на большую случайность или разнообразие в данных, тогда как низкая энтропия указывает на упорядоченные или предсказуемые данные.
