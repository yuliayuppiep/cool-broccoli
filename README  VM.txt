READ ME VIRTUAL MASHINE
Данный проект представляет собой простой интерпретатор для выполнения программ. 
Программа работает с памятью и выполняет различные операции, такие как ввод-вывод, арифметика, вычисление факториала и нахождение наибольшего общего делителя.

Программа считывает содержимое файла first.jpp и разбивает его на инструкции для выполнения.
Память программы представлена массивом mem, в котором хранятся значения и адреса для выполнения операций.
Инструкционный указатель ip управляет выполнением инструкций в программе.
Каждая инструкция выполняет определенную операцию, такую как ввод числа, вывод результата, арифметические операции и более сложные действия, включая вычисление факториала и нахождение наибольшего общего делителя.
Программа обрабатывает каждую инструкцию поочередно, увеличивая инструкционный указатель для перехода к следующей операции.

Пример программы в файле first.jpp:
python
Copy code
input 28
input 29
set 28 29 30
add 28 29 30
factorial 28 30
nod 28 29 30
jmp 28 29 0
jout 30
