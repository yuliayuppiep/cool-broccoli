README HAFFMAN
Этот код реализует алгоритм Хаффмана для кодирования и декодирования символов.
Алгоритм строит оптимальное префиксное дерево Хаффмана на основе частоты символов во входной строке.

tree представляет собой дерево Хаффмана, используемое для кодирования и декодирования символов.
Функция MinElementsIndex() находит индекс минимального неиспользованного вхождения в дереве для дальнейшего объединения узлов.
Этот код Хаффмана является эффективным методом сжатия данных и широко используется в сжатии файлов, передаче данных через сеть и других областях, где требуется оптимальное представление данных.
